# 기본 사용법 (주제만 입력)

가장 일반적인 실행 방법입니다.

## 사용법

```bash
cd ~/brain50/dev
./run.sh "주제 문장"
```

### 예시
```bash
./run.sh "오메가3가 정말 뇌에 좋을까?"
./run.sh "운동이 기억력에 미치는 영향"
```

## 동작
- JOB_ID는 자동으로 현재 시간 (`20250620_141500`)으로 생성됩니다.
- 결과는 `data/work/20250620_141500/` 폴더에 저장됩니다.
- 모든 중간 파일(script.txt, voice.wav, subs.srt, broll.mp4 등)이 이 폴더 안에 격리됩니다.

## 장점
- 간단하고 빠름
- 매번 파일을 수정할 필요 없이 주제만 바꿔서 실행 가능
- 현재 수동 확인 워크플로우에 최적화
